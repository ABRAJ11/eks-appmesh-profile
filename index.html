Flux CD

Flux is a Kubernetes controller that automatically ensures that the state of a cluster matches the config in git. Helm Operator is a Kubernetes CRD controller that manages the Helm release lifecycle.

App Mesh

AWS App Mesh is a service mesh that provides application-level networking to make it easy for your services to communicate with each other across multiple types of compute infrastructure.

Flagger

Flagger is a Kubernetes operator that automates the promotion of canary deployments using App Mesh routing for traffic shifting, Prometheus metrics for canary analysis and Helm for testing.
